# Rock Paper Scissors Game

A simple browser-based Rock Paper Scissors game built using HTML, CSS, and JavaScript.  
The player competes against the computer with real-time score updates and result messages.

UI screen :

## 🎮 Features

- Interactive Rock, Paper, Scissors selection
- Computer generates random moves
- Real-time game result display
- Score tracking (Player vs Computer)
- Clean and simple UI
- Fully client-side implementation

## 🎯 How the Game Works

- Player selects Rock, Paper, or Scissors
- Computer randomly selects one option
- Winner is decided based on rules:
  - Rock beats Scissors
  - Paper beats Rock
  - Scissors beats Paper
- Score updates automatically after each round
